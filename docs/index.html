Projetado para qualquer um, em qualquer lugar.

O Bootstrap torna o desenvolvimento front-end na web mais rápido e fácil. Ele é feito para pessoas de todos os níveis de conhecimento, e dispositivos de qualquer forma ou tamanho, e para projetos de qualquer tamanho.

Pré-processadores

O Bootstrap vem com CSS puro, mas seu código-fonte utiliza os dois pré-processadores CSS mais populares, Less e Sass. Comece a desenvolver rapidamente com seu CSS pré-compilado ou modifique o código-fonte.

Um framework,
qualquer dispositivo.

Com apenas um código-base, o Bootstrap adapta seus projetos web para telas de celulares, tablets e desktops de forma fácil e eficiente, através de media queries CSS.

Cheio de recursos

Com o Bootstrap, você tem uma linda e extensa documentação para elementos HTML comuns, dezenas de componentes CSS e HTML personalizados e incríveis plugins jQuery.

O Bootstrap tem código aberto. É hospedado, desenvolvido e mantido no GitHub.
