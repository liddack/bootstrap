O que está incluso

O Bootstrap está disponível para download em duas formas, nas quais você encontrará os seguintes diretórios e arquivos, agrupando logicamente recursos comuns e fornecendo ambas as variações: compilados e minificados.

jQuery obrigatório

Por favor, note que todos plugins JavaScript necessitam do jQuery para serem incluídos, como mostrado no template básico. Consulte nosso bower.json para ver quais versões do jQuery são suportadas.

Bootstrap pré-compilado

Feito o download, extraia a pasta compactada para ver a estrutura do Bootstrap (compilado). Você verá algo parecido com isto:

{% highlight bash %} bootstrap/ ├── css/ │ ├── bootstrap.css │ ├── bootstrap.css.map │ ├── bootstrap.min.css │ ├── bootstrap.min.css.map │ ├── bootstrap-theme.css │ ├── bootstrap-theme.css.map │ ├── bootstrap-theme.min.css │ └── bootstrap-theme.min.css.map ├── js/ │ ├── bootstrap.js │ └── bootstrap.min.js └── fonts/ ├── glyphicons-halflings-regular.eot ├── glyphicons-halflings-regular.svg ├── glyphicons-halflings-regular.ttf ├── glyphicons-halflings-regular.woff └── glyphicons-halflings-regular.woff2 {% endhighlight %}

Esta é a forma mais básica do Bootstrap: arquivos pré-compilados caem como uma luva em quase qualquer projeto web. Nós fornecemos CSS e JS compilados (bootstrap.*), assim como CSS e JS compilados e minificados (bootstrap.min.*). Source maps CSS (bootstrap.*.map) estão disponíveis para uso com ferramentas de desenvolvimento em determinados navegadores. Fontes Glyphicons estão incluídas, assim como o tema opcional do Bootstrapl.

Código-fonte do Bootstrap

O download do código fonte do Bootstrap inclui CSS, JavaScript e assets fonts, tudo pré-compilado, junto com código-fonte Less, JavaScripts e a documentação. Mais precisamente, ele inclui algo assim:

{% highlight bash %} bootstrap/ ├── less/ ├── js/ ├── fonts/ ├── dist/ │ ├── css/ │ ├── js/ │ └── fonts/ └── docs/ └── examples/ {% endhighlight %}

Os diretórios less/, js/, e fonts/ inclui os códigos-fonte do nosso CSS, JS, e fontes de ícones (respectivamente). O diretório dist/ sozinho já vem com tudo listado na seção de download pré-compilado. O diretório docs/ inclui o código-fonte da documentação, com exemplos de uso em examples/. Além disso, qualquer outro arquivo incluído fornece suporte para pacotes, informações de licença e desenvolvimento.
