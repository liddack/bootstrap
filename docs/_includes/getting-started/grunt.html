Compilando CSS e JavaScript

O Bootstrap usa o Grunt para seu sistema de compilação, com métodos práticos para trabalhar com o framework. É como compilamos nosso código, executamos testes, e muito mais.

Instalando Grunt

Para instalar o Grunt, você deve primeiro baixar e instalar o node.js (que inclui o npm). O npm auxilia com os pacotes de módulos do node e é uma maneira de gerenciar dependências de desenvolvimento através do node.js.

Então, a partir da linha de comando:
  1. Instale globalmente o grunt-cli com npm install -g grunt-cli.
  2. Navegue até a raiz do diretório /bootstrap/, e depois execute npm install. O npm vai identificar o arquivo package.json e automaticamente instalar as dependencias necessárias listadas neste arquivo.

Quando concluído, você será capaz de executar os vários comandos disponibilizados pelo Grunt a partir da linha de comando.

Comandos disponiveis no Grunt

grunt dist (Apenas compila o CSS e o JavaScript)

Regenera o diretório /dist/ com arquivos CSS e JavaScript compilados e minificados. Para usuários do Bootstrap, este é normalmente o comando mais desejado.

grunt watch (Observar)

Monitora em tempo real os arquivos fonte Less e automaticamente recompila para CSS sempre que você salvar alguma alteração.

grunt test (Executar testes)

Executa testes usando o JSHint e o QUnit em PhantomJS.

grunt docs (Compilar & testar os recursos da documentação)

Compila e testa o CSS, JavaScript, e outros assets que são usados ao executar a documentação localmente via jekyll serve.

grunt (Compilar absolutamente tudo e rodar testes)

Compila e minifica o CSS e JavaScript, constrói o site da documentação, executa o validador HTML5 na documentação, regenera recursos personalizados, e muito mais. Requer Jekyll. Normalmente só é necessário se você está explorando a fundo o Bootstrap.

Solução de Problemas

Se você encontrar problemas com a instalação de dependências ou ao executar comandos Grunt, primeiro exclua o diretório /node_modules/ gerado pelo npm. Então, execute novamente npm install.
