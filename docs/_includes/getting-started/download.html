Instalação com Bower

Você pode tambem instalar e manusear o Less, CSS, JavaScript e fontes do Bootstrap usando o Bower:

{% highlight bash %}$ bower install bootstrap{% endhighlight %}

Instalação com npm

Você também pode instalar o Bootstrap usando o npm:

{% highlight bash %}$ npm install bootstrap@3{% endhighlight %}

require('bootstrap') irá carregar todos os plugins jQuery do Bootstrap para dentro do objeto jQuery. O módulo bootstrap sozinho não exporta nada. Você pode carregar os plugins jQuery do Bootstrap manualmente e individualmente carregando os arquivo /js/*.js que estão dentro do diretório principal do pacote.

O arquivo package.json do Bootstrap contém alguns metadados adicionais sob as seguintes chaves:

  • less - caminho para o principal arquivo Less do Bootstrap
  • style - caminho para o CSS não minificado do Bootstrap, que foi pré-compilado usando as configurações padrão (sem personalização)

Instalação com Composer

Você pode tambem instalar e manusear o Less, CSS, JavaScript e fontes do Bootstrap usando o Composer:

{% highlight bash %}$ composer require twbs/bootstrap{% endhighlight %}

Less/Sass requerem Autoprefixer

O Bootstrap usa o Autoprefixer para trabalhar com prefixos CSS para diferentes browsers. Se você está compilando o Less/Sass do Bootstrap e não está usando o nosso Gruntfile, então você precisará integrar o Autoprefixer no seu processo de construção. Se você estiver usando Bootstrap pré-compilado ou usando o nosso Gruntfile, você não precisa se preocupar com isso porque o Autoprefixer já está integrado no nosso Gruntfile.
