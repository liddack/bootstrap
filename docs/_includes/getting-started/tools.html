Bootlint

O Bootlint é o linter de HTML oficial do Bootstrap. Ele automaticamente verifica seu código à procura de erros comuns de HTML em páginas que usam o Bootstrap em sua forma pura. Os componentes/widgets do Bootstrap puro devem ser encaixados dentro de certos tipos de estruturas do DOM. O Bootlint verifica se as instâncias de componentes do Bootstrap estão devidamente estruturadas. Considere adicionar o Bootlint às suas ferramentas de desenvolvimento com Bootstrap para evitar que esses erros comuns desacelerem o desenvolvimento do seu projeto.
